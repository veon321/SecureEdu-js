# User Management Panel

A sleek, modern, and fully interactive frontend User Management application that lets you manage users across different roles — admin, teacher, student, and guest. Built with vanilla technologies, it features a fluid dynamic layout, sophisticated dark UI elements, role-based panels, and smooth, intuitive interactions.
